# 🍉 さぼぴょん

かわいいサボテン女の子「サボてんちゃん」と一緒にジャンプして高さを競うゲームです！

## 🎮 ゲーム概要

「さぼぴょん」は、自民党の「安倍ぴょん」ゲームをベースにしたジャンプゲームです。2つのゲームモードで楽しめます！

### 🎯 ゲームモード

1. **⏱️ 1分タイムアタック**
   - 制限時間は1分間
   - 時間内にできるだけ高く登ってハイスコアを目指そう！
   - ハイスコアは自動保存されます

2. **♾️ エンドレス**
   - 時間無制限！
   - 落下するまで永遠に挑戦できます
   - どこまで高く登れるか挑戦しよう！

### 主な機能

✅ **完成している機能**
- **2つのゲームモード**: 1分タイムアタック＆エンドレス
- **BGM搭載**: ゲーム中に楽しい音楽が流れます
- **背景透過キャラクター画像**: PNG形式のサボてんちゃん（1.5倍サイズ）
- **ハイスコアシステム**: モード別にハイスコアを記録・表示
- **ハイスコア更新祝福**: 記録更新時に特別な画面と「🍻 乾杯！」メッセージ
- **強化されたジャンプ力**: 通常ジャンプ力が1.5倍
- **2倍ジャンプ特殊足場**: 🚀 黄色の足場で超高速ジャンプ
- **モバイル操作UI**: スマートフォン向けのジョイスティック
- プラットフォームに乗って上へ登る
- リアルタイムスコア表示（到達した高さ）
- ゲームオーバー時の結果表示
- スコアに応じたメッセージ表示
- レスポンシブデザイン（PC・スマホ対応）

### ゲームルール

1. **操作方法**
   - **スマホ/タブレット**:
     - ジョイスティックで左右移動（ドラッグ）
     - プラットフォームに着地すると自動ジャンプ
   - **PC**:
     - 左右キー: キャラクターを左右に移動
     - プラットフォームに着地すると自動ジャンプ
     - ジョイスティックも使用可能

2. **プラットフォームの種類**
   - **🟢 緑色の足場**: 通常ジャンプ（1.5倍の強化済み）
   - **🚀 黄色の足場**: 2倍ジャンプで超高速上昇！

3. **ゲームの流れ**
   - モード選択（1分タイムアタック or エンドレス）
   - スタートボタンを押してゲーム開始
   - プラットフォームに乗るたびに自動的にジャンプ
   - 黄色の足場でさらに高くジャンプ
   - タイムアタック: 1分以内にできるだけ高く
   - エンドレス: 画面下に落ちるまで挑戦
   - ハイスコア更新時は特別な祝福画面が表示！

4. **スコアリング**
   - 到達した高さがスコアになります（メートル単位）
   - 高く登るほど高得点！
   - 黄色の足場を活用して記録を伸ばそう！
   - ハイスコアはモード別に保存されます

## 🎮 遊び方のコツ

- プラットフォームの中央に着地すると安定します

## 🏆 ハイスコアについて

- ハイスコアはブラウザのLocalStorageに保存されます
- モード別（1分タイムアタック・エンドレス）に記録されます
- ハイスコア更新時は特別な祝福画面が表示されます
- 「🍻 乾杯！」でお祝いしましょう！
